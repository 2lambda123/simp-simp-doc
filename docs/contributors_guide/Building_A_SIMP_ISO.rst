Building a SIMP ISO
===================

The following guides provide information on building a SIMP ISO for yourself.

You may want to do this for various reasons including:

  * Building a :term:`RHEL` ISO
  * Customizing the kickstart
  * Adding your own packages
  * Updating packages from the defaults

.. WARNING::

    Prior to starting any build method, you will need to ensure that you follow
    the instructions in :ref:`gsg-environment_preparation`.
